# InfraHub Workspace Configuration

This workspace contains the codebase for the **InfraHub** Hardware & Network Services website.

## Rule Definitions

Rules are modularized and automatically loaded from the **`.claude/rules/`** directory. Refer to the specific rule files for detailed guidelines:

1. **[project-overview.md](file:///c:/Users/admin/OneDrive/Desktop/Sonny/Infrahub%20website/.claude/rules/project-overview.md):** Core identity, colors, design language (minimalist, clean, white modern), and workspace folder structure.
2. **[tech-stack.md](file:///c:/Users/admin/OneDrive/Desktop/Sonny/Infrahub%20website/.claude/rules/tech-stack.md):** Approved technologies (Vanilla HTML5, CSS3, JS ES6+), fonts (Outfit + Inter), and interactive component guidelines (map, cost estimator).
3. **[code-style.md](file:///c:/Users/admin/OneDrive/Desktop/Sonny/Infrahub%20website/.claude/rules/code-style.md):** CSS custom variables, spacing/radii definitions, layout rules, accessibility (unique IDs, heading hierarchy), and page performance standards.
4. **[agent-swarm.md](file:///c:/Users/admin/OneDrive/Desktop/Sonny/Infrahub%20website/.claude/rules/agent-swarm.md):** The Claude SWARM framework guidelines for scoping, wiring, assigning, running, and merging multi-agent task groups.
