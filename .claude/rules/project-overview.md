# InfraHub: Project Overview & Identity

This rule defines the core mission, aesthetic styling identity, and folder layout of the InfraHub website.

## Core Identity
InfraHub is a premium, high-performance network connectivity and hardware provisioning platform.
- **Design Inspiration:** Styled similarly to PacketFabric (bold technical layouts, clear grids) but configured as a clean "white modern" layout.
- **Aesthetic:** Minimalist, clean, white modern, professional.
- **Color Palette:**
  - Backgrounds: Pure White (#ffffff), Slate Gray (#f8fafc)
  - Typography: Deep Slate (#0f172a, #334155)
  - Accents: Electric Blue (#2563eb) and Tech Teal (#0d9488)
  - Layout: High whitespace, clean border-lines, thin grids.

## Directory Structure
Ensure all modifications preserve or align with the core workspace folder structure:
- `/index.html` - Primary HTML5 entry point and content document.
- `/index.css` - Custom styling library, variables, and responsive classes.
- `/index.js` - Client-side animation, map toggling, and configuration calculator logic.
- `/DESIGN.md` - Design Tokens manifest file.
- `/.claude/rules/` - Modular rule guides.
